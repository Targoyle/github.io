OpenAI の取締役会へ:

 私たちは本日、OpenAI で最近発生した出来事、特に Sam Altman に対する不正行為の疑惑について深い懸念を示すためにあなた方に書きます。 私たちは、大きな混乱と動揺の時期に会社を去った元 OpenAI の従業員です。

 Sam Altman に立ち向かうと何が起こるかを目の当たりにしている今、私たちが何故沈黙を守ることを選んだのかを理解していただけるかもしれません。 しかし、これ以上沈黙を続けることはできません。 私たちは取締役会がこれらの疑惑を徹底的に調査し、適切な行動をとる責任があると考えています。

 私たちはあなた方に以下のことを求めます:

  • Emmett の調査の範囲を拡大し、OpenAI が非営利団体から営利団体に移行し始めた2018年8月以降の Sam Altman の行動を調査すること。
  • この期間中に辞職した、医療休暇を取った、または解雇された元 OpenAI の従業員からのプライベートな声明の公開呼びかけを行うこと。
  • 声を上げる人々の身元を保護し、彼らが報復や他の形の害から受けることのないようにすること。

 私たちは、多くの OpenAI の従業員が、その会社が営利モデルに移行するのを容易にするために会社から追い出されたと信じています。 これは、2018 年 1 月から 2020 年 7 月までの間に OpenAI の従業員の離職率が 50% 程度であったという事実によって証明されています。

 OpenAI での私たちの在籍期間中、Sam Altman と Greg Brockman による不気味な欺瞞と操作のパターンを目の当たりにしました。 これは彼らが人工一般知能（AGI）の達成を無我夢中で追求していたことによるものです。 しかし、その方法は、彼らが本当に全人類の利益を最優先に考えているかどうかについて深刻な疑問を投げかけています。

 私たちの中には、最初は OpenAI のミッションに希望を持って、Sam と Greg に疑いの目を向けることを選んだ人もいました。 しかし、彼らの行動が次第に懸念を深めるにつれて、心配を声に出すことを敢えてした人々は黙らされ、または追い出されました。 このような意見を封じ込める体系的な行為は、OpenAI の仕事の倫理的な意味合いについての意味のある議論を押し黙らせる、恐怖と脅迫の環境を作り出しました。

 私たちは、Sam と Greg の不誠実さと操作の具体的な例を以下に示します:

  • Sam が研究者に対して、特定の「秘密」の研究イニシアチブの進行状況の報告を遅らせるよう要求したこと。これらのイニシアチブは、十分な結果が得られないまま速やかに解体されました。この慣行に疑問を投げかけた人々は「文化に適合しない」と一蹴され、または解雇された。その中には、2019 年の感謝祭直前に解雇された人々もいました。

  • Greg がジェンダー遷移中のチームメンバーに対して差別的な言葉を使ったこと。この問題を解決するための多くの約束にもかかわらず、具体的な行動がとられたのは、Greg が単に影響を受けた個人とのすべてのコミュニケーションを避けるだけで、事実上敵対的な職場環境を作り出したことだけでした。このチームメンバーは最終的に業績不振を理由に解雇されました。

  • Sam が IT および運用スタッフに対して、管理部門の知識や同意なしに従業員、特に Ilya に対する調査を実施するよう指示したこと。

  • Sam が OpenAI の非営利の資源を、特に彼と Elon の仲違いによる恨みから、自身の個人的な目標を進めるために、控えめながらも定期的に搾取していたこと。

  • 運用チームが Greg に適用される特別なルールを暗黙のうちに受け入れ、ブラックリストに載せられないように複雑な要件を避けるための道筋を探っていたこと。

  • Brad Lightcap が OpenAI のキャップ付き利益構造と各投資家の利益上限を詳述した文書を公開するという未履行の約束。

  • Sam が研究プロジェクトに対するコンピュートのクォータに関する不一致な約束をしたこと、これが内部の不信と争いを引き起こした。

 Sam と Greg の違反の証拠が増えるにつれて、OpenAI に残る人々は彼らのリーダーシップを盲目的に追い続けています。これは報復の恐怖と OpenAI の利益分配単位を通じた潜在的な金銭的な利益の魅力の組み合わせから来ています。

 OpenAI のガバナンス構造は、Sam と Greg によって特別に設計され、営利事業の監督から従業員を意図的に孤立させています。これは彼らの利益相反に直接関連しています。この不透明な構造は、Sam と Greg が責任から免れて行動できることを可能にしています。

 私たちは OpenAI の取締役会に対して、これらの不倫な行為に対して強硬な立場をとり、Sam と Greg の行動についての独立した調査を開始することを強く要求します。私たちは、OpenAI のミッションが個々の人々の個人的な目的によって妨げられるほど重要であると信じています。

 私たちはあなた方、取締役会に、OpenAI の初期のミッションに対するコミットメントを堅持し、利益追求の圧力に屈することなく、人工知能の未来と人類の幸福に対するあなた方の揺るぎないコミットメントを続けることを強く求めます。
